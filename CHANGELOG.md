## [1.0.0] - 2026-02-27

### Added

- Real-time VRM avatar lipsync via FFT audio analysis (60 fps)
- Webcam face tracking via OpenSeeFace (52 ARKit blendshapes)
- Text-to-speech with lip-synced avatar via KokoroSharp TTS bridge
- Expression system with 6 emotions + manual overrides
- Idle animation controller (breathing, micro-saccades, blinking)
- Gaze tracking and head pose correction
- VRM avatar library with download manager
- Hot-reloadable JSON tuning and mapping configuration
- CI with JSON schema validation and GDScript linting
- Landing page using @mcptoolshop/site-theme
- SECURITY.md with vulnerability reporting process and data scope
- SHIP_GATE.md and SCORECARD.md for product standards compliance
